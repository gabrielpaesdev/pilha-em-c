# Implementação de Pilha em C (LIFO)

Código-fonte desenvolvido ao vivo/passo a passo durante a vídeo-aula do canal. O projeto aborda a criação, manipulação e listagem de uma estrutura de dados de **Pilha (LIFO)** em C usando alocação estática.

🎥 **Assista à aula completa no YouTube:** [Clique Aqui](https://www.youtube.com/watch?v=BpYF5egx1_8)

---

## 📌 Conteúdo da Aula

Neste projeto e no vídeo, você vai ver na prática:
* O que é a estrutura de dados Pilha (LIFO - *Last In, First Out*);
* Como estruturar a pilha utilizando `struct` e arrays fixos;
* Implementação das funções `push` (empilhar) e `pop` (desempilhar);
* Tratamento de condições de **Pilha Cheia** e **Pilha Vazia**;
* Criação de um menu interativo no terminal (`main.c`).
